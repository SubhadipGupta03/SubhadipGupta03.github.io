## üì∏ Image Setup Guide

### Profile Picture
- Size: 300x300px (square)
- Format: JPG or PNG
- Name: `profile.jpg`
- Location: `assets/` folder
- Make it professional and clear

### Project Images
- Size: 600x400px (landscape)
- Format: JPG or PNG
- Names: `project1.jpg`, `project2.jpg`, `project3.jpg`
- Location: `assets/` folder
- Can be screenshots of websites or app interfaces

### Getting Images
- Use screenshots for web projects
- Use Photos app to resize images
- Or use online tool: https://pixlr.com/resize/
